RVV benchmark MegaBobcat

Bobcat is Tenstorrents open source fork of BOOM, which adds vector extension support. It supports a large subset of the ratified 1.0 version of the V vector extension including half precision floating points, and has a VLEN of 256 bits. The unimplemented instructions are: vdiv, vdivu, vfsqrt, vfsqrt7, vfrec7, vfdiv, vfrdiv, vremu, vrem. MegaBobcat is a configuration of Bobcat using MegaBoom, and 4 vector units.

Note that all of the measurements where using verilator to simulating the hardware.
